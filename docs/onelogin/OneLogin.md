# OneLogin (0.0.1)

OneLogin is a cloud-based identity and access management (IAM) platform that provides centralized authentication, management of user identities, and access control to applications and data.
OneLogin Events record logs related to an organization, such as user logins, password changes, and application access. The Events can be used to understand platform and user activity,  troubleshoot problems, and investigate security incidents.
The OneLogin Get Events API allows an organization to programmatically access the OneLogin Event logs. The API provides a way to retrieve, filter, and export events.

## Collections

## Get Events API (get_events_api) Collection

The OneLogin Get Events API provides near real-time, read-only access to an organization's event log.
