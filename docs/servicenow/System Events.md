# ServiceNow - System Events (0.0.1)

> Entity Name: event_source

ServiceNow system events are generated by pre-defined scripted triggers when users perform actions in the system.

## References
* [Event Logs Schema](https://docs.servicenow.com/bundle/washingtondc-platform-administration/page/administer/platform-events/reference/r_EventLogs.html)

## Retention

Based on our research, ServiceNow retains audit logs for Infinite.


### Comments
Can be changed by an instance admin.


## Latency

Based on our research, ServiceNow has a latency of Near Real-Time.

### Comments
Can vary based on system conditions and configurations.


## Licensing

Included with ServiceNow instances.
